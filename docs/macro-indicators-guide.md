# 宏观指标观测指南

> 替代自建环境分类模型（模块四），用市场上成熟的公开指标判断资金多空与市场环境。
>
> 核心思路：**不预测涨跌，只判断当前环境适合多大仓位。**

---

## 一、货币与流动性（月频，决定大方向）

这组指标决定市场的"水位"——钱多了资产涨，钱少了资产跌。

### M2 同比增速

- **是什么**：广义货币供应量的同比增长率，反映央行放了多少水
- **怎么看**：
  - M2 增速 > 前值且 > 8% → 流动性宽松，利好权益
  - M2 增速连续下降 → 流动性收紧，谨慎
  - M2 - 社融增速差（剪刀差）扩大 → 资金淤积在金融体系，可能推升资产价格
- **数据源**：央行官网 / 东方财富宏观数据 / Wind
- **频率**：每月 10-15 日公布上月数据

### 社会融资规模

- **是什么**：实体经济从金融体系获得的资金总量
- **怎么看**：
  - 社融增速拐头向上 → 信用扩张，经济预期改善，领先股市 1-2 个季度
  - 社融增速持续下滑 → 信用收缩，防御为主
  - 重点看新增社融中"企业中长期贷款"占比 → 高说明企业投资意愿强
- **数据源**：央行官网 / 东方财富
- **频率**：每月公布

### Shibor / DR007（银行间利率）

- **是什么**：银行间市场的短期资金价格，反映资金面松紧
- **怎么看**：
  - DR007 持续低于政策利率（7天逆回购利率）→ 资金面宽松
  - DR007 大幅高于政策利率 → 资金面紧张，短期承压
  - 月末/季末/年末季节性上升属正常波动
- **数据源**：中国货币网 / 东方财富
- **频率**：每日，但月度趋势更有意义

### 10年期国债收益率

- **是什么**：无风险利率的锚，影响所有资产定价
- **怎么看**：
  - 收益率下行 → 利好股市（折现率降低，资金从债市流向股市）
  - 收益率快速上行 → 利空成长股，高估值承压
  - 10Y-1Y 期限利差收窄/倒挂 → 经济衰退预期
- **数据源**：中国债券信息网 / 东方财富
- **频率**：每日

## 二、资金流向（周频，判断短中期多空力量）

### 融资融券余额

- **是什么**：杠杆资金的规模，反映场内投资者的做多/做空意愿
- **怎么看**：
  - 融资余额持续增加 → 场内加杠杆做多，情绪偏乐观
  - 融资余额快速飙升至历史高位 → 过热信号，注意风险
  - 融资余额持续下降 → 去杠杆，市场偏弱
  - 融券余额突增 → 有资金在做空，关注
- **数据源**：交易所官网 / 东方财富 / 同花顺
- **频率**：每日公布，看周度趋势

### 北向资金（沪深港通）

- **是什么**：外资通过港股通流入 A 股的资金
- **怎么看**：
  - 连续多日净流入且金额大（>50亿/日）→ 外资看多
  - 连续净流出 → 外资避险，可能有全球风险事件
  - ⚠️ 注意：北向资金中有不少是量化对冲盘，单日数据噪音大，看周度/月度趋势
- **数据源**：东方财富 / 同花顺
- **频率**：每日，看周度趋势

### 新发基金规模

- **是什么**：公募基金新发行的规模，反映居民入市意愿
- **怎么看**：
  - 爆款基金频出、单周新发 > 500 亿 → 市场过热，可能见顶
  - 新发基金持续低迷 → 市场底部区域的特征之一
  - 这是一个反向指标：大家都在买的时候要小心
- **数据源**：Wind / 天天基金网
- **频率**：周度

## 三、估值水位（月频，判断贵不贵）

### 全 A 股 PE/PB 分位数

- **是什么**：当前估值在历史中的位置（过去 10 年）
- **怎么看**：
  - PE 分位 < 30% → 低估区域，可以积极
  - PE 分位 30%-70% → 合理区域，正常配置
  - PE 分位 > 70% → 高估区域，逐步减仓
  - PB 分位更适合周期股和银行
- **数据源**：乌龟量化 / 理杏仁 / 东方财富
- **频率**：每日更新，月度决策

### 股债性价比（风险溢价）

- **是什么**：全 A 股盈利收益率（1/PE）- 10年期国债收益率
- **怎么看**：
  - 风险溢价 > 历史均值 + 1 标准差 → 股票极具吸引力
  - 风险溢价 < 历史均值 - 1 标准差 → 股票太贵，债券更优
  - 这是巴菲特指标的中国版，长期有效
- **数据源**：乌龟量化 / 自行计算
- **频率**：月度

## 四、市场情绪（日/周频，辅助择时）

### 换手率

- **是什么**：市场交易活跃度
- **怎么看**：
  - 全 A 换手率 > 2.5% → 交易过热
  - 全 A 换手率 < 0.8% → 地量见地价，关注反转
  - 持续缩量阴跌 → 还没到恐慌性抛售，可能还没见底
- **数据源**：东方财富 / baostock
- **频率**：每日

### 涨跌家数比 & 涨停跌停数

- **是什么**：市场广度指标，反映赚钱效应
- **怎么看**：
  - 上涨家数 / 下跌家数 > 3 → 强势普涨
  - 上涨家数 / 下跌家数 < 0.3 → 恐慌性下跌
  - 指数涨但多数个股跌（二八分化）→ 行情末期特征
- **数据源**：东方财富 / 同花顺
- **频率**：每日

### VIX（中国波指 iVIX / 50ETF 期权隐含波动率）

- **是什么**：市场恐慌程度
- **怎么看**：
  - iVIX < 15 → 市场平静，低波动环境
  - iVIX 15-25 → 正常波动
  - iVIX 25-35 → 波动加大，减仓
  - iVIX > 35 → 恐慌，但也可能是抄底机会（需结合其他指标）
- **数据源**：上交所 / 东方财富期权页面
- **频率**：每日

## 五、观测框架：如何综合判断

不要看单一指标，用"投票制"综合判断：

### 仓位决策矩阵

| 环境 | 流动性 | 估值 | 情绪 | 建议仓位 |
|------|--------|------|------|---------|
| 全面利好 | 宽松 | 低估 | 低迷/回暖 | 70-80% |
| 偏多 | 宽松 | 合理 | 正常 | 50-70% |
| 中性 | 中性 | 合理 | 正常 | 40-60% |
| 偏空 | 收紧 | 合理/高估 | 过热 | 20-40% |
| 全面利空 | 收紧 | 高估 | 恐慌 | 0-20% |

### 观测节奏

| 频率 | 关注指标 | 动作 |
|------|---------|------|
| 每日（扫一眼） | DR007、北向资金、换手率、涨跌家数 | 异常才关注 |
| 每周（5分钟） | 融资余额趋势、新发基金、VIX | 更新情绪判断 |
| 每月（15分钟） | M2/社融、PE分位、股债性价比 | 调整仓位中枢 |
| 每季（30分钟） | 利率周期、信用周期拐点 | 调整大方向 |

### 关键原则

1. **流动性是最大的基本面** — M2 和社融的方向比任何个股分析都重要
2. **不要和央行作对** — 货币宽松周期做多，收紧周期防守
3. **极端值比趋势更有用** — 指标到极端位置时的信号最可靠
4. **情绪指标是反向的** — 所有人都乐观时最危险，所有人都绝望时最安全
