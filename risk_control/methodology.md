## 一、仓位管理（第一道防线）

**依据来源：** 沪深300指数的历史波动率（HV20）

```
建议仓位上限 = max(100% - 2× HV, 20%)
举例：HV=23.1% → 上限=100%-46.2%=53.8% → 取整60%
```

**建议格式：**

```
仓位: 72%  建议≤60%  超出12%

```

## 二、止损止盈（第二道防线）

### 2.1 止损价

**公式：**
```
止损价 = max(成本价, 当前价) - ATR倍数 × ATR(14)
ATR = Wilder平滑真实波幅（周期14）
参数：STOP_LOSS_ATR_MULTIPLIER = 2.0
```

**触发条件：** 当前价 ≤ 止损价 → 🔴止损

### 2.2 移动止损（适用于盈利仓位）

**公式：**
```
移动止损触发价 = 近14日最高价 - 1.5 × ATR(14)
参数：TRAILING_STOP_ATR_MULTIPLIER = 1.5
```

**触发条件：** 当前价 ≤ 移动止损价（且持仓盈利）→ 🟠移动止损

### 2.3 分批止盈

**参数表：**
| 盈利幅度 | 卖出比例 | 说明 |
|---------|---------|------|
| +15% | 1/3 | 第一批止盈 |
| +30% | 1/3 | 第二批止盈 |

**触发条件：** 当前价触及止盈档位 → 🟡止盈

**参数：**

|------|------|
| 日 | -3% |
| 周 | -5% |

## 四、异常检测（第三道防线）

| 异常类型 | 判断标准 | 阈值 |
|---------|---------|------|
| 波动率突变 | 当日HV / 历史HV均值 | >2.5倍 |
| 流动性枯竭 | 量比（当日量/5日均量） | <0.3 |
| 相关性过高 | 持仓间Pearson相关系数 | >0.85 |

## 五、操作建议格式规范

**每条建议必须包含：**
1. 操作动作（加仓/减仓/止损/止盈/持有）
2. 具体标的
3. 计算依据（数值来源 + 公式 + 结果）
4. 参考参数值

**示例：**
```
🔴 互联网ETF(159792) 已触及止损
依据: 成本0.8736 → 止损价0.841(ATR×2) → 现价0.666 < 止损价
建议: 执行止损，减仓60% → 保留40%观望
```
